
Msg2Clip プラグイン
================================================================================

メールのヘッダビューにクリップボードへファイルをコピーするための
ボタンを追加するSylpheed向けのプラグインです。

動作環境
--------------------------------------------------------------------------------

Windows XP SP3

Sylpheed 3.2.0beta5以降

インストール方法
--------------------------------------------------------------------------------

msg2clipをSylpheedのプラグインディレクトリへとコピーします。

1) Sylpheedのzipアーカイブを使用している場合

pluginsディレクトリへ配置します。

2) Sylpheedをインスーラによりセットアップした場合

%APPDATA%/Sylpheed/plugins以下に配置します。
Windows XPの場合以下の場所にdllをコピーして下さい。
C:\Documents and Settings\(ユーザアカウント)\Application Data\Sylpheed\plugins

1)もしくは2)によりセットアップは完了です。
問題なければ ステータスバーに アイコン が表示されるはずです。

使い方
--------------------------------------------------------------------------------

メールを表示したときに表示されるヘッダビューに表示されている
ボタンをクリックするとメール本文のファイルがクリップボードへと
コピーできます。

いったんクリップボードへとコピーできてしまえが、
あとはファイラやエクスプローラなどで任意の場所へと貼り付けることができ
るようになります。

注意事項
--------------------------------------------------------------------------------

Win32 APIを使用しているため、Windows 環境以外では動作しません。
